Description

Carve your path. Create your journey.
Journey is a simple resource to help you plan your next trip whether you know where to go or not.
Select your destination or Journey can do it for you. Select dates, travel method, overnight accomodations and preferred restaurants or types of attractions. Journey will then populate nearby results for you.

Technologies

Bcrypt - Used for user password hashing
Express - Middleware for routing HTTP requests
Express-session - Used for storing user session information for authentication
Google Places Autocomplete API - Google's API for autocompleting geo searches and populating results
Heroku - Application and database hosting
MongoDB(Atlas) (Mongoose) - Database storage for user accounts and trips
NodeJS - Javascript runtime environment for the application
Passport.js - Used for log-in authentication
React - FrontEnd Framework
SweetAlert - NPM package for custom alerts
